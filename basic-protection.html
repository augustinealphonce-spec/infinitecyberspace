Basic Protection for SMEs

Day 1: Password Security 🔑

  • Use at least 12 characters
  • Mix letters, numbers, and symbols
  • Avoid personal info
  • Encourage password managers

👉 Task: Update one business account with a stronger password today.

Day 2: Antivirus Setup 🛡️

  • Install reputable antivirus software
  • Enable automatic updates
  • Run weekly scans

👉 Task: Install or update antivirus on your office computers.

Day 3: Safe Browsing 🌐

  • Don’t click unknown links
  • Verify email senders
  • Use HTTPS websites

👉 Task: Share one suspicious email example with your team.

Day 4: Software Updates ⚙️

  • Enable auto-updates for OS and apps
  • Patch vulnerabilities quickly

👉 Task: Check if your systems are up-to-date today.

Day 5: Backup Basics 💾

  • Schedule regular backups
  • Use cloud + external drives
  • Test recovery process

👉 Task: Perform a test backup and recovery this week.
