🔐 Advanced Security

DIY steps to strengthen your network and data protection.

Network Protection

  • Change your Wi-Fi password every 3 months
  • Use WPA3 or WPA2 encryption on your router
  • Disable remote management unless needed

👉 Task: Log in to your router and update the Wi-Fi password today.

Data Protection

  • Enable device encryption (BitLocker, FileVault, Android/iOS built-in)
  • Use strong, unique passwords for each account
  • Turn on two-factor authentication (2FA)

👉 Task: Enable 2FA on your email and social media accounts.

Backup & Recovery

  • Schedule weekly backups to an external drive
  • Use cloud storage for critical files
  • Test restoring a file at least once a month

👉 Task: Back up your documents folder to Google Drive or OneDrive.

Safe Browsing

  • Install browser security extensions (HTTPS Everywhere, uBlock Origin)
  • Clear cookies and cache regularly
  • Check website URLs before entering personal info

👉 Task: Install a trusted ad-blocker extension in your browser.

Device Hygiene

  • Update operating system and apps regularly
  • Uninstall unused software
  • Run antivirus scans weekly
